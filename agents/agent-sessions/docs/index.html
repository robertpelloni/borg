Agent Sessions
• macOS • Open Source • ⭐ –

Session Browser for Codex CLI, Claude Code, Gemini CLI, GitHub Copilot CLI & Droid

Agent Sessions 2 brings Codex CLI, Claude Code, Gemini CLI, GitHub Copilot CLI, and Droid (Factory CLI) together in a single, powerful interface. Look up any past session — even the ancient ones /resume can't show — or browse visually to find that lost perfect prompt or code snippet, then instantly copy or resume what you need.

local-first, open source, and built for terminal vibe warriors.

Install with Homebrew Tap

brew tap jazzyalex/agent-sessions
brew install --cask agent-sessions

6 Agents Supported

Browse all your agent sessions in a unified window. Enable only your favorite agents and then filter sessions by agent instantly.

  • Codex CLI
  • Claude Code
  • OpenCode
  • Factory Droid CLI
  • GitHub Copilot CLI
  • Gemini CLI

Unified Search

Search across all agents instantly. Filter by message type (User prompt, Tool call, Error) in Color View. Smart sorting, project filters, and cancellable full-text search.

Instant Resume

Reopen any Codex or Claude session in Terminal/iTerm2. Browse visually and copy snippets when you need them.

Saved Sessions

New dedicated Saved Sessions window for managing archived sessions. Delete, reveal, and diagnose with archive status tooltips.

Usage Tracking

Always-on tracking of 5-hour and weekly limits for Codex and Claude. Menu bar and in-app indicators show usage with auto-refresh probes.

Analytics

Interactive flip cards with sparklines and insights. Session trends, agent breakdowns, and time-of-day heatmaps. Git Inspector shows repository state.

Local & Private

All processing on your Mac. Reads session directories locally. No cloud uploads or telemetry. Read-only by design.
